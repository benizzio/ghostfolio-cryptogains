# Bug Report: Project Actions Lack Makefile Entry Points

**Type**: Spec gap
**Severity**: Medium
**Feature**: `002-sync-data-validation`

## Description

The reported requirement is that project actions should be exposed through a `Makefile` so contributors can execute the slice consistently through stable entry points.
