# Changelog

All notable changes to this extension are documented in this file.

## [0.0.0] - 2026-07-04

### Added

- Initial Spec Kit extension manifest.
- `prepare` command for adding or refreshing a context-orchestration work-unit ledger in the active feature `tasks.md`.
- `validate` command for read-only pre-implementation ledger checks.
- Strict prerequisite validation requiring `None` or earlier `WU##` work-unit references.
- Optional `after_tasks` hook and required `before_implement` validation hook.
- Documentation for the reverse-engineered task-file pattern.
